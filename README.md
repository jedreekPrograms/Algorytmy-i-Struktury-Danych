# Algorithms and Data Structures

Collection of laboratory assignments completed during the **Algorithms and Data Structures** course at **Wrocław University of Science and Technology (Politechnika Wrocławska)**. The repository includes implementations of fundamental data structures, classic algorithms, and experimental complexity analysis. :contentReference[oaicite:0]{index=0}

## Laboratory 1 – Basic Data Structures
Implementation of stacks, queues, and cyclic linked lists with performance analysis of search operations.

## Laboratory 2 – Sorting Algorithms
Implementation and experimental comparison of classic and hybrid sorting algorithms, including complexity analysis and efficient matrix search.

## Laboratory 3 – Selection Algorithms
Implementation of Randomized Select, Median of Medians (SELECT), optimized searching techniques, and experimental performance evaluation.

## Laboratory 4 – Trees and Graph Algorithms
Implementation and analysis of Binary Search Trees, Splay Trees, Minimum Spanning Tree algorithms (Prim and Kruskal), and information propagation algorithms for trees.
